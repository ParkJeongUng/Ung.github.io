지도학습 // 비지도학습

  • X 데이터를 모델이 분석해서 Y 데이터를 예측한다
  • 우리가 구해야 할 Y 데이터는 타겟 값이라고한다.
  • 지도학습 --> 입력(input)데이터와 타깃(target)데이터 2가지를 합친 훈련 데이터를 가지고있다. // 모델훈련때 X 데이터 Y 데이터 둘다 필요
  • 비지도학습 --> 타깃데이터 없이 입력데이터만 사용한다. // 모델훈련때 X 데이터만 필요

훈련세트 // 테스트세트

  • 훈련 세트 --> 모델의 훈련을 위해 사용되는 데이터
  • 테스트 세트 --> 훈련된 모델의 평가에 사용되는 데이터

그리드 서치

  • 자동으로 최적의 하이퍼 파라미터를 찾는다
from sklearn.model_selection import GridSearchCV
dt = DecisionTreeClassifier(random_state = 42)

# 딕셔너리 형태로 내가 원하는 파라미터의 범위를 설정
params = {'max_depth' : [1, 2, 3, 4]}

# 객체설정
gs = GridSearchCV(dt, param_grid = params, n_jobs = -1)
gs.fit(X_train, Y_train)

GridSearchCV(estimator=DecisionTreeClassifier(random_state=42), n_jobs=-1,
             param_grid={'max_depth': [1, 2, 3, 4]})
dt = gs.best_estimator_
